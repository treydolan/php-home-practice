Two Required + Two Defaulted

Call with defaults:

Item: Studio Monitors
Qty: 2
Discount: 0%
Note: (no note)
Since there are no parameter values for discount and note, they display their defaults.

Call overriding defaults:

Item: Audio Interface
Qty: 1
Discount: 15%
Note: Student discount
Discount and note are overridden by parameter values.

Pass by Value vs Reference

Before

Array
(
    [volume] => 70
    [balance] => 0
    [bass] => -1
    [treble] => 1
)

After

Array
(
    [volume] => 70
    [balance] => 0
    [bass] => 1
    [treble] => 4
)

volume/balance unchanged (by value); bass/treble modified (by reference).
